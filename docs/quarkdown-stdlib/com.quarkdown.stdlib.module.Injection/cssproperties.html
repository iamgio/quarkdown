cssproperties

.cssproperties properties:{Map<String, Any>} -> Node
Content copied to clipboard

Overrides the value of Quarkdown CSS properties. Each entry corresponds to a Quarkdown CSS property name and its value. The names will be prefixed with --qd- to match the Quarkdown CSS variable naming convention.

For example:

.cssproperties
  - background-color: green
  - main-font-size: 20px
Content copied to clipboard

For a complete list of properties, see the global theme. Unknown properties will be ignored.

The content is wrapped in a <style> tag and rendered as-is, without any additional processing or escaping, as long as the rendering target supports HTML.

Return

a new Html node representing the style element

Parameters

properties

a map of CSS property names and their values
