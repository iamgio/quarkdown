Parameters

width

height

height of the container

fullWidth

whether the container should take up the full width of the parent. Overridden by width

foregroundColor

text color

backgroundColor

borderColor

border color

border width

borderStyle

border style

margin

whitespace outside the content

cornerRadius

border radius of the container

alignment

alignment of the content

textAlignment

alignment of the text

textTransform

transformation applied to the text content

fontSize

relative font size of the text

float

floating position of the container within the subsequent content

className

custom class name for the container, useful for applying custom styles, if supported by the renderer
