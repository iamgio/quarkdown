AfterAllRenderingPeek

object AfterAllRenderingPeek : PeekPipelineStage<CharSequence>

Peek stage that runs after all rendering has been completed.

  • It usually matches with PipelineHooks.afterPostRendering.

  • If post-rendering is disabled by com.quarkdown.core.pipeline.PipelineOptions.wrapOutput set to false, it will match with PipelineHooks.afterRendering instead.

This stage allows for consistently hooking into the final rendered output, regardless of whether post-rendering is enabled or not.
