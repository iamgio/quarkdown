LexingStage

class LexingStage(source: CharSequence) : PipelineStage<Set<Library>, List<Token>>

Pipeline stage responsible for lexical analysis (tokenization) of the input text.

This stage produces a list of tokens as output from the provided source text. It uses the lexer factory from the context's flavor to create a block lexer that processes the source text and breaks it down into tokens.

The tokens produced by this stage are used by the ParsingStage to build an abstract syntax tree.
