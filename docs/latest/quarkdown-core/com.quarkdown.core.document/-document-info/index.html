data class DocumentInfo(val type: DocumentType = DocumentType.PLAIN, val name: String? = null, val description: String? = null, val authors: List<DocumentAuthor> = emptyList(), val keywords: List<String> = emptyList(), val locale: Locale? = null, val numbering: DocumentNumbering? = null, val theme: DocumentTheme? = null, val tex: TexInfo = TexInfo(), val layout: DocumentLayoutInfo = DocumentLayoutInfo())

Immutable information about the document. This data is updated by library functions .docname, .docauthor, etc., by overwriting com.quarkdown.core.context.MutableContext.documentInfo.
